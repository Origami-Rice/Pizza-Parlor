# PizzaParlour

Foobar is a Python library for dealing with word pluralization.

## Installation

follow the requirement.txt, pip3 install them

## Usage

Run the main Flask module by running
```bash
python3 PizzaParlour.py
```

then run the app by
```bash
python3 Main.py
```

## Test
Run the unit tests by
```bash
python3 Main.py
```
